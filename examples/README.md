# مثال‌های استفاده از فریمورک WeWork

این پوشه شامل مثال‌های مختلف استفاده از فریمورک WeWork است.

## ساختار

```
examples/
├── basic/              # مثال‌های پایه
│   ├── simple-api/     # API ساده
│   ├── crud/           # CRUD کامل
│   └── form/           # فرم با Validation
│
├── advanced/           # مثال‌های پیشرفته
│   ├── real-time/      # Real-time با WebSocket
│   ├── file-upload/    # آپلود فایل
│   └── search/         # جستجو و فیلتر
│
└── integration/        # یکپارچه‌سازی‌ها
    ├── sms/            # یکپارچه‌سازی SMS
    ├── payment/        # یکپارچه‌سازی پرداخت
    └── telegram/       # یکپارچه‌سازی Telegram
```

## استفاده

هر مثال شامل:
- کد کامل Backend و Frontend
- فایل README با توضیحات
- فایل‌های تنظیمات
- دستورالعمل اجرا

برای مشاهده مستندات کامل مثال‌ها، به [مستندات مثال‌ها](../docs/06-examples/basic-examples.md) مراجعه کنید.
