# Trading Bot Strategy Builder UI

رابط کاربری برای ساخت و تست استراتژی‌های معاملاتی با استفاده از drag-and-drop

## نصب و راه‌اندازی

### پیش‌نیازها
- Node.js 18+ و npm

### نصب
```bash
cd frontend
npm install
```

### اجرا
```bash
npm run dev
```

رابط کاربری در آدرس `http://localhost:3000` در دسترس خواهد بود.

### ساخت برای تولید
```bash
npm run build
```

فایل‌های ساخته شده در پوشه `dist` قرار می‌گیرند.

## استفاده

1. **ساخت استراتژی جدید**: روی "ساخت استراتژی جدید" کلیک کنید
2. **افزودن اندیکاتور**: از پنل سمت راست، اندیکاتور مورد نظر را انتخاب کنید
3. **اتصال گره‌ها**: گره‌ها را با کشیدن خط بین آن‌ها به هم متصل کنید
4. **تنظیم پارامترها**: روی هر گره کلیک کنید و پارامترهای آن را تنظیم کنید
5. **ذخیره استراتژی**: نام استراتژی را وارد کرده و روی "ذخیره استراتژی" کلیک کنید
6. **بک تست**: از لیست استراتژی‌ها، روی "بک تست" کلیک کنید و تنظیمات را وارد کنید

## ساختار

- `src/App.jsx` - کامپوننت اصلی
- `src/components/StrategyBuilder.jsx` - سازنده استراتژی با drag-and-drop
- `src/components/StrategyList.jsx` - لیست استراتژی‌های ذخیره شده
- `src/components/BacktestPanel.jsx` - پنل بک تست
- `src/components/IndicatorPalette.jsx` - پالت اندیکاتورها
- `src/components/NodePanel.jsx` - پنل تنظیمات گره
- `src/components/StrategyConfig.jsx` - تنظیمات استراتژی
