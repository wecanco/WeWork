# UI Kit و کامپوننت‌ها

برای مستندات کامل UI Kit، به [مستندات UI Kit](../04-ui-kit/introduction.md) مراجعه کنید.

## استفاده سریع

```jsx
import { Button, Card, Input } from '../ui'

function MyComponent() {
  return (
    <Card>
      <Input label="نام" />
      <Button variant="primary">ذخیره</Button>
    </Card>
  )
}
```
