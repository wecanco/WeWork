# مرجع کامل API

این بخش مرجع کامل API های فریمورک WeWork را ارائه می‌دهد.

## Authentication API

- POST `/api/auth/register` - ثبت‌نام
- POST `/api/auth/login` - ورود
- GET `/api/auth/me` - اطلاعات کاربر فعلی
